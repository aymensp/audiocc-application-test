## Quick use


``` bash

# install project
git clone https://github.com/aymensp/audiocc-application-test.git
cd calculator
npm install
npm run dev

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# run unit tests
npm run unit

```
